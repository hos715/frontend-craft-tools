# Contributing

Thanks for considering contributing to frontend-craft-tools!

Quick guide to add a new tool:

1. Create a folder under `public/tools/<tool-name>/`.
2. Add an HTML page (`<tool-name>.html`) and a script (`<tool-name>.js`) that run client-side only.
3. Add any styles under `public/styles.css` or a local CSS file.
4. Document the tool in `docs/tools/<tool-name>.md` and add an entry to the README.

Please open a Pull Request and include screenshots or a short demo of the tool.
