# JSON Minifier

ब्राउज़र-आधारित JSON फ़ाइलों को कम करने के लिए एक सरल उपकरण।

## अवलोकन

यह प्रोजेक्ट एक स्थैतिक HTML पेज प्रदान करता है जो उपयोगकर्ताओं को JSON फ़ाइल को ड्रैग-और-ड्रॉप या चयन करने की अनुमति देता है, फिर बिना किसी सर्वर के मिनिफ़ाइड संस्करण डाउनलोड करता है।

## विशेषताएं

- JSON फ़ाइल ड्रैग-और-ड्रॉप समर्थन
- फ़ाइल चयन बैकअप
- मिनिफिकेशन से पहले और बाद का आकार तुलना
- मिनिफ़ाइड JSON आउटपुट डाउनलोड
- पूरी तरह से क्लाइंट-साइड, कोई बैकएंड नहीं
- बहु-भाषा समर्थन: فارسی, English, 中文 और हिंदी
- GitHub Pages के लिए तैयार

## उपयोग

1. अपने ब्राउज़र में `index.html` खोलें।
2. JSON Minifier टूल लिंक पर क्लिक करें।
3. `.json` फ़ाइल को ऐप पर ड्रैग करें या फ़ाइल क्षेत्र पर क्लिक करके चुनें।
4. फ़ाइल प्रोसेस होने के बाद डाउनलोड बटन पर क्लिक करें।

- `lottie-color-switcher.js` — Lottie रंग बदलने वाला पेज स्क्रिप्ट
