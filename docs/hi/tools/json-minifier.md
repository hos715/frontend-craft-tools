# JSON Minifier

ब्राउज़र-आधारित JSON फ़ाइलों को कम करने के लिए एक सरल उपकरण। फ़ाइल को ड्रैग करें या चुनें, फिर संकुचित आउटपुट डाउनलोड करें।

## उपयोग

1. JSON Minifier टूल पेज खोलें।
2. एक `.json` फ़ाइल ड्रैग या चुनें।
3. संकुचित परिणाम डाउनलोड करें।
