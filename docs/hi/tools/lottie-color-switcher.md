# Lottie Color Switcher

Lottie JSON फ़ाइल से यूनिक रंग निकालें और उन्हें बदलें (Hex या RGBA)। यह उपकरण ब्राउज़र में पूरी तरह से चलती है और अपडेटेड JSON डाउनलोड करती है।

## उपयोग

1. Lottie Color Switcher टूल पेज खोलें।
2. एक Lottie JSON फ़ाइल अपलोड करें।
3. रंग बदलें और लागू करें।
4. संशोधित फ़ाइल डाउनलोड करें।
