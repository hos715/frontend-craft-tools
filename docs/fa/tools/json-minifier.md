# JSON Minifier

یک ابزار سمت کاربر برای فشرده‌سازی فایل‌های JSON. می‌توانید فایل را بکشید و رها کنید یا انتخاب کنید و خروجی فشرده‌شده را دانلود کنید.

## روش استفاده

1. صفحه ابزار JSON Minifier را باز کنید.
2. فایل `.json` را بکشید یا انتخاب کنید.
3. نتیجهٔ فشرده‌شده را دانلود کنید.
