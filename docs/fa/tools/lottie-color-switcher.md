# Lottie Color Switcher

رنگ‌های منحصربه‌فرد را از فایل Lottie JSON استخراج کنید و آن‌ها را با مقدار هگز یا RGBA جایگزین کنید. ابزار به صورت کامل در مرورگر اجرا می‌شود و JSON به‌روزشده را دانلود می‌کند.

## روش استفاده

1. صفحه ابزار Lottie Color Switcher را باز کنید.
2. فایل Lottie JSON را بارگذاری کنید.
3. رنگ‌ها را تغییر دهید و تغییرات را اعمال کنید.
4. فایل اصلاح‌شده را دانلود کنید.
